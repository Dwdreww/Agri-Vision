About

About AGRI-VISION

AGRI-VISION is a pineapple crop health scanner connected to a Flask backend using your YOLO and EfficientNet models.

YOLO Stage

YOLO scans the uploaded image and finds possible anomaly regions using bounding boxes.

EfficientNet Stage

Each YOLO crop is passed into EfficientNet for disease classification.

Frontend Output

The frontend shows the annotated image, prediction class, confidence, and detection count.
